Misc Game UI & Menu sound FX

■概要

Misc Game UI & Menu sound FXはハイクオリティかつパワフルな
ゲームでよく使われるメニューやUI周辺の効果音パッケージです。

200ファイルを超えるオリジナルサウンドで構成され、それらは決定音、セレクト音
アイテム取得音等のゲーム制作でよく使われる13のわかりやすいカテゴリに
分類され、wavフォーマットで格納されています。

このパックを使う事で手軽にゲームの基本となる音を揃える事ができます。


■内容説明

総数　247 Sounds - 44.1K 16bit Wav Stereo
